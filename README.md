# UK Research Position Explorer

Interactive explorer for analysing UK research positioning across 4,516 topics using OpenAlex data (2010-2024).

## Views

- **Topic Browser** - Search and filter topics by field, trajectory pattern, and bloc context
- **Topic Detail** - Deep dive with trajectory charts and country comparison
- **Bloc Comparison** - Analysis against three major science blocs (Europe, USA, China)
- **Country Analysis** - Comparison across 46 countries
- **UK Strengths Dashboard** - Where UK research is gaining momentum

## Data

Analysis based on OpenAlex scholarly publication data. Compares UK research share trends against:
- Historical periods (2010-2017 vs 2016-2024)
- Three major science blocs (Europe, USA, China)
- 46 comparator countries

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
